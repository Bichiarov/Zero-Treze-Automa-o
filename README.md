# Zero13 Automação Comercial

Site estático pronto para publicar no GitHub Pages.

## Estrutura

- `index.html` na raiz do repositório
- pasta `assets/` na raiz do repositório

## Como publicar

1. Envie o arquivo `index.html` para a raiz do repositório.
2. Envie a pasta `assets` inteira.
3. Faça o commit.
4. No GitHub, vá em `Settings > Pages`.
5. Configure `Deploy from a branch`, branch `main`, pasta `/root`.
6. Aguarde o deploy e atualize o navegador com `Ctrl + F5`.
